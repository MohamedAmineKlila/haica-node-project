# HAICA Internship Project

A full-stack web application developed during my internship at HAICA using Node.js, Fastify, Vue 3, and MySQL.

## Features

- User authentication
- User management
- Role-based access control
- RESTful API
- Frontend built with Vue 3
- MySQL database integration
- Secure backend using Fastify
- Responsive user interface

## Tech Stack

### Backend
- Node.js
- Fastify
- MySQL
- JWT Authentication
- REST API

### Frontend
- Vue 3
- JavaScript
- HTML
- CSS

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/your-repository.git
cd your-repository
```

Install dependencies:

```bash
npm install
```

Configure your database credentials in the backend configuration file.

## Running the Application

Start the backend:

```bash
cd backend
npm install
npm run dev
```

Start the frontend:

```bash
cd frontend
npm install
npm run dev
```
